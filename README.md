> Comprehensive TensorFlow examples for both v1.x and v2.x.

Tutorial collection covering linear regression, logistic regression, Word2Vec, CNNs, RNNs (LSTM), autoencoders, DCGANs, multi-GPU training, TensorBoard, and more.

---

## Topics Covered

**Basic Models**: Linear regression, Logistic regression, Word2Vec, GBDT
**Neural Networks**: Simple NN, CNN, RNN (LSTM), Bi-directional RNN
**Generative Models**: AutoEncoder, DCGAN
**Utilities**: Save/restore models, Custom layers, TensorBoard
**Data Management**: TFRecords, Image transformation, Multi-GPU
